HomRec

Documentation

Complete guide to using HomRec screen recorder.


Installation

Option A — Portable .exe (recommended)

  1. Download the latest .zip from Download page
  2. Unzip anywhere you want — no installer needed
  3. Download ffmpeg.exe from ffmpeg.org
  4. Place ffmpeg.exe in the same folder as hr.exe
  5. Launch hr.exe

Option B — Run from source

  1. Clone the repo: git clone https://github.com/homaaio/HomRec.git
  2. Install dependencies: pip install -r requirements.txt
  3. Place ffmpeg.exe in the HomRec folder or add to PATH
  4. Run: python homrec.py

Quick Start Guide
  1. Launch HomRec — double-click hr.exe
  2. Select recording area — choose full screen or specific window
  3. Choose audio sources — microphone, desktop audio, or both
  4. Press F9 to start recording
  5. Press F9 again to stop recording
  6. Find your video in the output folder

Keyboard Shortcuts
F9Start / Stop recording
F10Pause / Resume recording
F11Toggle fullscreen mode

Main Features
  • Screen recording — full desktop or specific window
  • Audio capture — microphone + desktop audio (WASAPI loopback)
  • Multi-monitor — select which monitor to record
  • PC Analytics — CPU, RAM and disk usage in one window
  • Hotkeys — customizable keyboard shortcuts
  • Languages — English and Russian UI (add more with .hrl files)
  • Catppuccin theme — dark UI inspired by Catppuccin Macchiato
  • Custom output folder — choose where recordings are saved
  • Recording stats — live FPS, duration, frame count in status bar
  • Always on top — keep the window above everything else
  • System tray — minimise to tray, control recording from tray menu
  • Auto update check — notifies you when a new version is out

Audio Setup

HomRec can record two types of audio:

  • Microphone — your voice via connected mic
  • Desktop audio — system sounds, game audio, music, etc. (WASAPI loopback)

Important: Desktop audio capture requires pyaudio-wasapi instead of standard PyAudio. If you're running from source, install it with:

pip install pyaudio-wasapi

For the portable .exe version, desktop audio is already supported out of the box.

FFmpeg Setup

HomRec requires FFmpeg for video encoding. To set it up:

  1. Go to ffmpeg.org/download.html
  2. Download the Windows build (e.g., ffmpeg-release-full.7z)
  3. Extract the archive and find ffmpeg.exe inside the bin folder
  4. Copy ffmpeg.exe to the same folder as hr.exe

Alternatively, add the FFmpeg bin folder to your system PATH.

Adding Languages

HomRec uses .hrl (HomRec Language) files for translations. To add a new language:

  1. Open an existing .hrl file (e.g., en.hrl) in a text editor
  2. Translate each string to your target language
  3. Save the file with the language code (e.g., es.hrl for Spanish)
  4. Place the file in the languages folder next to hr.exe
  5. Restart HomRec and select your language from the settings menu

Send us your translation and we'll include it in the next release!

System Requirements
  • OS: Windows 7 / 8 / 10 / 11
  • RAM: 1 GB minimum (2 GB recommended)
  • Storage: 350 MB free space
  • No GPU required — works on any graphics hardware

Troubleshooting

Antivirus warning: Add HomRec folder to exceptions.

No desktop audio: Ensure you have pyaudio-wasapi installed.

FFmpeg not found: Place ffmpeg.exe next to hr.exe.

Recording lags: Lower the frame rate or resolution in settings.

Can't find recordings: Check the output folder in settings menu.
