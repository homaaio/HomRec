HomRec

Screen recorder built for weak PCs.
No lags. No bloat. No GPU.

If you've ever tried recording with OBS on an old laptop and everything lagged — HomRec is made for you. Written from the ground up to use minimum CPU and RAM. Just open, record, done.

Setup
Get running in
two minutes.
Option A — .exe (recommended)
No Python or FFmpeg setup required. Everything is bundled in the archive.
1
Go to Releases and download the latest .zip
2
Unzip anywhere — no installer needed
3
Launch !hr.exe
Option B — from source
Requires Python 3.8+ and FFmpeg on PATH or in the project folder.
# clone
git clone https://github.com/homaaio/homrec.git
cd homrec

# install deps
pip install -r requirements.txt

# run
python homrec.py
