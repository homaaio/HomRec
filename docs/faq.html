Frequently Asked Questions

Answers to common questions about HomRec screen recorder.


What is HomRec?
HomRec is a free and open source screen recorder designed specifically for weak PCs. It uses minimal CPU and RAM, requires no GPU, and has no background services. Just open, record, done.

Is HomRec really free?
Yes! HomRec is completely free software with open source code under the GNU LGPL license. You can use it on any computer, including commercial organizations, without registration or payment.

Why does my antivirus flag HomRec?
Some antiviruses (Kaspersky, Avast, etc.) may flag HomRec because it is a new program that is not yet widely known. It is not a virus - the full source code is available on GitHub for inspection. Add the HomRec folder to exceptions if needed.

Do I need FFmpeg?
No. HomRec uses FFmpeg in download folders (.zip/.7z). You don't need put it into folder "HomRec" but you can update FFmpeg if you want

Does HomRec record desktop audio?
Yes! HomRec captures both microphone and desktop audio using WASAPI loopback. This requires the pyaudio-wasapi library instead of standard PyAudio. If you're running from source, install it with: pip install pyaudio-wasapi

What are the keyboard shortcuts?
  • F9 - Start / Stop recording
  • F10 - Pause / Resume recording
  • F11 - Toggle fullscreen mode

Does HomRec work on Windows 7?
Yes! HomRec works on Windows 7, 8, 10, and 11. However, note that Windows 7 may require additional updates for WASAPI loopback support. For best results, use Windows 10 or 11.

How do I add more languages?
HomRec uses .hrl (HomRec Language) files. You can create your own translation by copying an existing language file and translating the strings. Send us your translation and we'll include it in the next release!

Where are recordings saved?
By default, recordings are saved in the same folder as hr.exe. You can change the output folder in the settings menu.

How can I report a bug or suggest a feature?
You can report issues and suggest features on our GitHub Issues page. You can also reach out via Telegram: @homaexe

Is there a portable version?
Yes! The .zip archive is completely portable. No installation required - just unzip and run hr.exe. All settings are saved in the same folder.

Why is the file size large?
The download size is around 310 MB because it includes all necessary dependencies (Python interpreter, OpenCV, PyAudio, etc.) bundled into a standalone executable. This ensures HomRec works right away without requiring Python to be installed.
