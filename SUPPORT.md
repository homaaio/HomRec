## Topics You Can Reach Out About

- Bug reports
- Feature requests and suggestions
- Installation and setup help
- If the program works on a PC below the system requirements
- Any other questions about HomRec

---

## How to Write a Good Bug Report

To help me resolve your issue quickly, please include:

1. **HomRec version**
2. **Windows version** (10/11, 64-bit or 32-bit)
3. **Brief description of the problem**
4. **Steps to reproduce** (how to trigger the bug)
5. **Screenshot or log** (if possible)

---

## Response Time

I usually reply within **1–3 days**. Please be patient — this is an open-source project, and I work on it in my free time.

---

## Security

- **Never include** personal passwords, serial numbers, or other sensitive data in Issues or emails.
- If you find a security vulnerability, please email me directly instead of opening a public Issue.
